Bootstrap the application
-------------------------

You have to add those files in your project.

* [app/config/core.php](https://github.com/cakephp/cakephp/blob/master/app/Config/core.php)
* [app/config/database.php](https://github.com/cakephp/cakephp/blob/master/app/Config/database.php.default)
* [app/config/email.php](https://github.com/cakephp/cakephp/blob/master/app/Config/email.php.default)
